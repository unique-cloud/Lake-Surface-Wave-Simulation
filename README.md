# Lake-Surface-Wave-Simulation

Implemented in serveral ways including

| Framework | Device |
|-----------|--------|
| Cuda | Nvidia GPU |
| Cuda + MPI | distributed Nvidia GPUs |
| OpenMP | multi-core CPUs |
| OpenAcc | cpu + gpus |
| Perm | |
| Horovod | distributed gpus |
